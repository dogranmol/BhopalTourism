Variables

Using Sass, you declare the variable name with $ in front, and then assign it a value (not a property, but a value, ie #F5F2D0).
I tend to use the practice of assigning all variables globally in a _variables.scss file,
making it incredibly easy to play with different colors through the application while maintaining consistency.
